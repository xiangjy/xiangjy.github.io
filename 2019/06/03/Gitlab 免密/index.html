Gitlab 免密

1、创建一个 SSH key

$ ssh-keygen -t rsa -f xxxx -C “xiangjy@outlook.com”

代码参数含义：

-t 指定密钥类型，默认是 rsa ，可以省略。

-C 设置注释文字，比如邮箱。

-f 指定密钥文件存储文件名。

生成 id_rsa 和 id_rsa.pub 两个秘钥文件。

2、添加SSH key 到 Gitlab

$ clip < ~/.ssh/id_rsa.pub

Account settings -> SSH key
